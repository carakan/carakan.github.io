Lanzamos nuestra empresa de desarrollo de paginas web

Carlos Ramos

October 1, 2007

Estamos contentos ya que desde ahora ofrecemos desarrollo de paginas web, además hacemos una evaluación de sitios ya existentes para que puedan tener ese sabor que recientemente se esta propagando por Internet, la web 2.0, una red de redes mas social, también estamos viendo la parte del desarrollo con ruby on rails a tope, ahora solamente nos queda trabajar duro :]

Quiero agradecer el apoyo de la Sra. Anne Marie Van Swinderen que nos esta solventando el hosting y el nombre de dominio muy generosamente, nuestra empresa es:

[](http://carakan.com/wp-content/uploads/2007/10/wiebia.jpg “Empresa de desarrollo de soluciones web 2.0”)

Bueno para poder hacer andar el servidor, por ahora es un VPS pero cuando se pueda nos moveremos al dedicado, me tuve que matar configurándolo, el servicio lo obtuvimos de RailsPlayground, un trato optimo para desarrollo con ruby on rails.

Esperamos que con estas iniciativas podamos de aquí a cierto tiempo estar a la par del desarrollo español líder a nivel de los hispanohablantes.
