Libero mi primer proyecto grande en Rails: Tematic

PUBLISHED ON APR 23, 2009

Después de un año tengo ahora la alegría de entregar este primer proyecto que realice, el cual esta desarrollado en Rails 2.0, el cual por ese entonces se necesito hacer un hibrido entre PHP y Rails, funcionando a la perfección, la url del repositorio GItHub es:

Proyecto Tematic.

La idea principal de este proyecto es la de crear un sistema de foros, noticias y chat dentro del trabajo de periodistas, el trabajo quedo discontinuado, pero pretendo reanudarlo enfocando la necesidad de acá, Bolivia, en el cual se necesitan las herramientas necesarias para los periodistas.
