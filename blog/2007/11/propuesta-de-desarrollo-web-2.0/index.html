Propuesta de desarrollo web 2.0

PUBLISHED ON NOV 16, 2007

Despues de bastante tiempo de estar viendo el mercado dentro de bolivia, sale esta propuesta de desarrollo de sitios web anexando lo que vendria a ser los Blogs y/o el concepto de Web 2.0, dentro de Wiebia con la colaboracion de Ramiro Huayller y ademas de la colaboracion de Anne Marie Van Swinderen, ha salido a la luz publica la propuesta de desarrollo de sitios web 2.0, tanto para empresas, comunidades y personas emprendedoras, con precios accesibles a la realidad boliviana. Lo pueden encontrar en cotizacion de sitios web 2.0 Un saludo.
