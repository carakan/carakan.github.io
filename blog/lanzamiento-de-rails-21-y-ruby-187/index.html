Lanzamiento de Rails 2.1 y ruby 1.8.7

Carlos Ramos

June 2, 2008

Parece que la RailsConf 2008 nos tiene traera bastantes sorpresas ya que anuncian el lanzamiento oficial de la version 1.8.7 del lenguaje de programacion ruby, que esta lista para bajar desde aqui. Ademas se lanza oficialmente la tan esperada version 2.1 de Rails, que creo esperaron para lucirse en la RailsConf 2008, si quieres aprender/saber sobre las nuevas caracteristicas de Rails que ofrece las encontraras en : Rails Cast para Rails 2.1. Y por ultimo se anuncia la version 2.0 de Passanger, que soporta ahora Rack. Tantas buenas noticias para un lunes :D
